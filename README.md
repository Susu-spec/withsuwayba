# With Suwayba

A personal portfolio website built with React and TailwindCSS.

## Tech Stack

- React (with TypeScript)
- Tailwind CSS


## Getting Started
```bash
  # Clone the repo
  git clone https://github.com/Susu-spec/withsuwayba.git
  cd withsuwayba

  # Install dependencies
  npm install

  # Run the development server
  npm run dev

```
